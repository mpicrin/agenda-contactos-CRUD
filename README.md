# 📒 Agenda de Contactos en Python / Python Contact Book

Este proyecto es una **Agenda de Contactos** desarrollada en Python utilizando **Programación Orientada a Objetos (POO)**.  
Permite realizar operaciones básicas de un CRUD:

- ➕ Agregar contactos  
- 🔍 Buscar contactos (con coincidencias parciales en el nombre)  
- 🗑️ Borrar contactos por índice  
- 📋 Mostrar todos los contactos  

---

## 🚀 Características / Features
- Clase **Contactos**: representa cada contacto con nombre y teléfono.  
- Clase **Agenda**: gestiona la lista de contactos y aplica validaciones.  
- Validación de teléfono: solo acepta números de al menos 8 dígitos.  
- Menú interactivo en consola con opciones claras.  
- Búsqueda flexible: encuentra coincidencias parciales en nombres.
